# Gemini Chatbot

Simple chatbot interface built with React and Vite on the frontend, and Express.js on the backend. The chatbot uses the Google AI studio API and can recognize speech using the Web Speech API

## Getting Started

1. Clone the repository:

```sh
git clone https://github.com/shumakooov/ai-chat.git
```

2. Install dependencies for both the frontend and the backend in the root folder:
```sh
npm install
```
3. Set up environment variables:
```sh
GOOGLE_GENAI_API_KEY=your_genaiapi_key
```
4. Run the server and client with one command from the root folder:
```sh
npm run dev
```
